# solidity-template

Basic setup to get started writing smart contracts.

## One-time development setup

```
PATH=$PATH:node_modules/.bin
npm install
truffle compile
```

In a tab, run:

```
npm run ganache
```

Then run:

```
truffle migrate
```

## Running tests

In a tab, run:

```
npm run ganache
```

Then run test cases with:

```
npm run test
```
